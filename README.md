##### 1. Clone ***( using `git clone` )*** or Download the folder ***( using ***`Download ZIP`*** option )*** #####

##### 3. Open the terminal and install all required dependencies with ***`npm i`*** #####
##### 4. type ***`npm start`*** to the app #####
